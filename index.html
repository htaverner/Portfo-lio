Installing htop on Linux

Overview


htop is a real-time process monitoring tool for Unix-like systems. It provides a dynamic, colorful, and interactive alternative to the classic top command.



Installation Steps


  1. Open your terminal.
  2. Run the appropriate installation command for your Linux distribution:

  • Ubuntu / Debian: 
    sudo apt update
sudo apt install htop
  • Fedora: 
    sudo dnf install htop
  • Arch Linux: 
    sudo pacman -S htop


Usage


Once installed, launch htop with:

htop

You’ll see a live view of system processes, CPU, and memory usage.
