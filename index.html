Welcome to My Portfolio

I'm Harry Taverner – a technical writer with a background in language, design, and storytelling. This is where I bring those skills together.

Tech Writing Work

I create user-centered documentation that balances clarity, tone, and structure. My approach is grounded in academic research and built for real-world users.

My academic background includes a PhD in language, stylistics, and discourse analysis, with a focus on how people process information and meaning. Combined with practical experience in HTML, CSS, Markdown, and tools like GitHub and Visual Studio Code, I build documentation that is accurate, accessible, and engaging.

Key Strengths:

  • Clear procedural writing
  • Visual documentation (tables, diagrams, infographics)
  • Working knowledge of Python, R, HTML/CSS
  • Git-based version control workflows
  • Plain language and style guide alignment

I'm currently building a set of sample docs, walkthroughs, and guides to demonstrate my approach across use cases. These will be added here soon.

Graphic Design Work

Although I no longer work professionally in graphic design, I still apply those skills regularly—especially in support of technical documentation.

I have experience producing clean, purposeful graphics to support understanding: infographics, interface mockups, icon sets, diagrams, and more.

My Toolkit includes:
