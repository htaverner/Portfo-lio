Install htop on Linux

Overview:

htop is an interactive process viewer for Unix systems. It is a more user-friendly alternative to top, offering colorized output and intuitive navigation.


Supported Systems: Ubuntu / Debian, Fedora, Arch Linux



Step-by-Step Instructions:


  1. Open your terminal.
  2. Run the appropriate installation command:

    • Ubuntu / Debian:
      sudo apt update
      sudo apt install htop
    • Fedora:
      sudo dnf install htop
    • Arch Linux:
      sudo pacman -S htop

  3. Launch htop to verify it works: htop


Troubleshooting:


  • If you see "command not found", update your package list: sudo apt update
  • On Ubuntu, try enabling the Universe repository if problems persist: sudo add-apt-repository universe

Tip: You can press F10 to quit htop.
